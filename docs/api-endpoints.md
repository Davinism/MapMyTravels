# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### Routes

- `GET /api/routes`
- `POST /api/routes`
- `GET /api/routes/:id`
- `PATCH /api/routes/:id`
- `DELETE /api/routes/:id`

### Trips

- `GET /api/trips`
- `POST /api/trips`
- `GET /api/trips/:id`
- `PATCH /api/trips/:id/`
- `DELETE /api/trips/:id`

### Tags

- `GET /api/trips/:id/comments`
- `POST /api/trips/:id/comments`
